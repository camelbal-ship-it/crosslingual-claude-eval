# crosslingual-claude-eval

## What This Is
A research project evaluating Claude's response consistency across six languages 
(English, French, Spanish, German, Italian, Romanian) using prompts that are 
semantically identical across all languages.

## Why It Matters
Cross-lingual AI output variation has real implications for equitable access, 
language education, and NLP benchmarking. This project applies ILR-informed 
analysis to identify and document those patterns systematically.

## Methodology
- Identical prompts administered across all six languages
- Outputs analyzed for consistency, nuance variation, and structural differences
- ILR/OPI framework applied to assess language-level response calibration

## About
Conducted by Camelia Baluta — six-language professional, former DLIFLC faculty, 
ILR/OPI assessment specialist, and AI adoption practitioner.
